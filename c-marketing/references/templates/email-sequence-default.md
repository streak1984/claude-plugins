# Email Sequence Campaign Template

Use this skeleton when generating email sequences in campaign mode. The existing `templates.md` has sequence type patterns — this template adds campaign-mode defaults.

## Campaign Launch Sequence (default)

When part of a campaign, default to a **launch sequence** (5-7 emails) unless the brief specifies otherwise.

## Structure per email

**Delay:** {timing}
**Subject:** {under 50 chars}
**Preheader:** {under 90 chars, extends subject}

### Body structure
1. Hook — 1-2 sentences, connects to reader's situation
2. Context — 1 short paragraph, why this matters now
3. Value — the meat, adapted to email's job
4. CTA — one action, benefit-driven
5. PS — optional, secondary link or softer ask

## Campaign-specific rules

- Pull tone and CTA patterns from client profile
- Reference other campaign content (article, newsletter) as CTA destinations
- Build sequence arc that aligns with the campaign's core message from the brief
- Use the brief's "Støttepunkter" as fodder for individual emails
