# Claude Plugins

Collection of Claude Code / Cowork plugins for content marketing and agency work.

## Plugins

| Plugin | Description |
|--------|-------------|
| [claude-everywhere](./claude-everywhere/) | Content marketing suite — campaigns, articles, ads, social, newsletters. Norwegian Bokmål default. |

## Installation

### From this repo

```bash
claude plugin install --from streak1984/claude-plugins/claude-everywhere
```

### Local development

```bash
git clone https://github.com/streak1984/claude-plugins.git
claude --plugin-dir ./claude-plugins/claude-everywhere
```

## Prerequisites

- **Pexels API key** (free) for stock images: `export PEXELS_API_KEY=your-key`
